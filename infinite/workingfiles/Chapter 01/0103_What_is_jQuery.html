JavaScript problems

Most browsers:

var xhr = new XMLHttpRequest();

IE6:

var xhr = new ActiveXObject("Microsoft.XMLHTTP");

JavaScript solutions

Code around the problem:

if (XMLHttpRequest) {
	var xhr = new XMLHttpRequest();
}
else {
	var xhr = new ActiveXObject("Microsoft.XMLHTTP");
}

jQuery is a library

Written in JavaScript for JavaScript developers.

  • a consistent API
  • powerful animation and page manipulation
  • easier event handling
  • simplifies Ajax functionality
  • useful utility functions
  • encourages extendible and reusable code
  • reduces the code you need to write

jQuery is a library

It's NOT:

  • a new language
  • a framework
  • cheating
  • strictly necessary
  • the only solution
  • perfect
