Efficient jQuery selectors

Use an ID if possible

$("#idname");

Avoid selecting by class

$(".myclass");
$("p.myclass"); // faster

Keep it simple!

Overly complex selectors take longer to parse…

$("body page:first-child article.main p#intro em");
$("p#intro em");

Be more specific on right

Selecting works from right to left:

$("p#intro em");
$("p#intro").find("em");
$("em", $("p#intro"));

Avoid selector repetition

$("p").css("color", "blue");
$("p").text("Text changed!");
// use chaining
$("p").css("color", "blue").text("Text changed!");
// or cache
var p = $("p");
p.css("color", "blue");
p.text("Text changed!");
