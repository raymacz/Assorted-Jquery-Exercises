Web browser events

An introduction

What is "an event"?

any user interaction with the web page

Such as?…

  • opening the page
  • hovering over an element
  • clicking an element
  • focusing a form field
  • typing a character
  • submitting a form
  • scrolling the page
  • resizing the window
  • closing the page

Step 1: event registration

  1. Identify the element
  2. Identify the action
  3. Register a handler

Step 2: create an event handler

  • a function
  • passed an event object
  • analyse the event object
  • perform the relevant actions

Step 3: further actions?

  • deregister the event
  • enable or disable further event actions

Multiple events

Any number of events can be applied to the same element.

Different event handlers can be registered for the same element and event type.
